# MLB Release Notes

Release: MLB Production Complete v1.0.0
Status: Production Stable / Maintenance Mode

## Summary

MLB is certified production complete for Pick Analyzer v1.0.0. The platform includes a stable dashboard, Current Board, official/informational category separation, provider-aware operating-day lifecycle, feature-store integration, model versioning guardrails, settlement/replay/learning contracts and documented provider limitations.

## Completed

- Dashboard unification and premium Today experience.
- Current Board intelligence engine.
- Official Picks, AI Leans, Watchlist and Avoid category separation.
- Most Likely, Best Value, AI Bet Finder, Betting Workbench and Arbitrage read paths.
- Operating-day lifecycle, next-slate rollover and automation status.
- SportsDataIO provider budget, capability and market audits.
- MLB odds coverage diagnostics.
- Starter/weather/stadium intelligence.
- Player metadata cache and roster availability health.
- Missing-intelligence and data-quality health.
- Feature Store integration.
- Prediction versioning, V6/V7 challenger isolation and model-ops guardrails.
- Settlement core and scoped settlement contracts.
- Historical replay, calibration and learning readiness surfaces.
- BSN transition path started after MLB stabilization.
