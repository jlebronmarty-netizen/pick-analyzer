# AI Prediction History UI

The Performance page loads prediction history independently from the main hero so slow history loading does not block the page.

Filters:

- Mode: All, Official, Shadow
- Result: All, Correct, Incorrect, Push, Pending
- Category
- Minimum confidence

Rows show date, sport, matchup, prediction, probability, confidence, result and official/shadow badges. Opening a row shows pregame prediction, actual result, stored feature context and evaluation details when available.

Historical predictions are not rewritten.
