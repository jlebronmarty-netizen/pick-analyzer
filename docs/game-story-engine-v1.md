# Game Story Engine V1

Game Story Engine V1 lives inside AI Sports Analyst V2 and turns stored Game Intelligence facts into a concise deterministic narrative.

## Sections

1. Matchup summary.
2. Model view.
3. Market view.
4. Main uncertainty.
5. Bottom line.

Numeric claims are emitted only when a stored source field exists. Otherwise the story reports unavailable context instead of using vague unsupported claims.
