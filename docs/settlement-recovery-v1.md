# Settlement Recovery V1

Production settlement remains deterministic-only.

The recovery planner classifies pending rows as final-result eligible, legacy, fixture/test, post-start, mapping-missing, result-missing, or legitimately pending. On 2026-07-22 the live planner reported no production-scoped rows eligible for deterministic settlement; recent MLB pending rows were model preview/test-like scope rather than official production settlement backlog.

No settlement should occur without exact event identity, exact market, exact line where applicable, exact selection, and final score/status evidence.
