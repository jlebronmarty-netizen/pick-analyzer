# MLB Features

Status: Production Stable
Version: MLB Production Complete v1.0.0

## Supported Production Features

- Dashboard Today contract and premium User Mode.
- Current Board official/informational separation.
- Official Picks governed by frozen policy thresholds.
- AI Leans, Watchlist and Avoid as non-official market intelligence categories.
- Most Likely, Best Value, AI Bet Finder, Betting Workbench and Arbitrage read paths.
- MLB operating-day lifecycle and next-slate rollover.
- Full-game moneyline, run line/spread and total market support where verified odds exist.
- Feature Store contracts and MLB feature-store integration.
- Starter/weather/stadium evidence from verified GamesByDate payloads.
- Player metadata cache and roster-status availability health.
- Missing-intelligence and data-quality health reporting.
- Provider capability audit and market capability registry.
- V6/V7 comparison, promotion-readiness, rollback-plan and shadow-evaluation surfaces without promotion.
- Settlement core and scoped settlement contracts.
- Learning and calibration readiness surfaces.

## Intentionally Non-Production Or Deferred

- V7 champion promotion.
- New prediction models.
- New official thresholds.
- Player props, pitcher props, batter props, NRFI/YRFI, first-five and team totals.
- Confirmed lineup confidence beyond explicit provider availability.
- Detailed injury diagnosis/severity/return dates.
- Multi-book arbitrage without verified simultaneous multi-book prices.
- Durable feature snapshot persistence beyond existing approved tables/contracts.

## Certification

MLB feature scope is complete for v1.0.0. Future MLB work should be maintenance mode only.
