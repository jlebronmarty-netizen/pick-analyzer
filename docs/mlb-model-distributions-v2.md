# MLB Model Distributions V2

## Status

Not activated for production recommendation decisions in this patch.

The current corrective work focused on cron reliability and data-quality honesty. Moneyline, run-line and total probabilities remain generated by the existing prospective preview model, with strict production, quarantine, calibration, confidence, edge and EV gates unchanged.

## Required V2 Direction

Future MLB V2 model work should add deterministic distribution tests before changing probabilities:

- moneyline probability bounds and home/away symmetry
- American odds implied probability and EV
- run-line expected margin distribution and line-specific cover probability
- total run distribution with over/under complementarity
- integer total push handling
- confidence components separated from probability
- missing starter, lineup, weather and bullpen penalties

Do not route totals or run lines through a moneyline-only transformation. Do not tune thresholds to create picks.
