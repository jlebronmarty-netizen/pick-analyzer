# Pitcher Outs Feature Contract

Pitcher recorded-outs features are built only from stored, timestamp-safe evidence.

## Implemented V1 Features

- recorded outs last 3
- recorded outs last 5
- recorded outs last 10
- season average
- median
- standard deviation
- threshold rates for 15+, 16+, 17+, 18+, 19+, 20+ and 21+
- starter evidence status
- feature quality
- data sufficiency

## Unsupported Until Verified

- real pitcher prop line
- sportsbook over/under prices
- verified MLB injury return flag
- confirmed lineup impact
- verified weather
- multi-book market movement

Unsupported inputs remain explicit missing features and must not be fabricated.
