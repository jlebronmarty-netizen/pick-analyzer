# AI Report Card

Daily AI Report Card V1 is informational only and does not activate betting.

Dimensions:

- Prediction Quality
- Calibration
- Confidence
- Data Quality
- Feature Quality
- Learning Progress
- Stability
- Provider Readiness
- Recommendation Readiness
- Overall Health

Grades are `A+`, `A`, `A-`, `B+`, `B`, `B-`, `C`, `D` and `INSUFFICIENT DATA`. Strong grades are blocked by inadequate samples.
