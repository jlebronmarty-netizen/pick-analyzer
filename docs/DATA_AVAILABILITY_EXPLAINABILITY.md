# Data Availability Explainability V1

User-facing copy groups domains into:

- Available
- Pending
- Unsupported

Required language is now represented by Adaptive Operations:

- Detailed injury information is unavailable under the current provider plan.
- Availability impact is limited because lineup confirmation is unavailable.
- Missing odds means recommendations are not actionable yet.

The system must not fabricate diagnosis, expected return, day-to-day status or confirmed lineup absence.
