# MLB Projection Board

The MLB Projection Board is available at `/projections`.

It shows sportsbook-independent shadow projections ordered by evidence strength:

1. Valid projections only.
2. Highest rank score.
3. Readiness.
4. Confidence.
5. Earliest upcoming game.

Sections:

- Top Projections.
- Pitchers.
- Teams.
- Batters.
- Games.

The board does not show sportsbook lines, Over/Under labels, EV, Kelly, Official Picks or betting recommendations.
