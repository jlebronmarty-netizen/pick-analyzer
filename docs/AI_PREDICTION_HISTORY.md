# AI Prediction History

Universal prediction history is sourced from stored `prediction_history` plus supported shadow replay evidence.

Supported filters:

- Sport
- League
- Date
- Model version
- Category
- Official / Shadow
- Correct / Incorrect / Push / Pending
- Confidence range

Rows expose date, matchup, prediction, probability, confidence, category, model version, actual result and settlement status.

Opening a prediction may show pre-game feature snapshot, missing data, final result, settlement details and an outcome explanation only when supported by stored evidence.

Historical predictions are never rewritten by AI Brain.
