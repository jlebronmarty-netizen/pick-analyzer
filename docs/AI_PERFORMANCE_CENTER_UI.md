# AI Performance Center UI

AI Performance Center Product UI V1 makes the existing AI Brain visible at `/performance`.

## Navigation

- Dashboard sidebar: Performance
- Dashboard header: Performance
- User Mode preview card: View AI Performance
- Performance page primary nav: Today, Performance, MLB, BSN, Advanced

## Page Structure

The page shows AI Brain Hero, All Sports, sport-specific view, Evolution, Daily Report Card, Trust Score, Why Trust Changed, Model Maturity, Goals, Prediction History, Timeline, Engineering Advisor and collapsed Internal Diagnostics.

All values come from existing `/api/performance` and `/api/performance/history` contracts.

Provider calls added: 0.

Database mutations added: 0.
