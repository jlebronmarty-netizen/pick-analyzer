# AI Performance Internal View

Internal AI Brain diagnostics are available through Advanced Details and API contracts.

Internal metrics include:

- Brier Score
- Log Loss
- Calibration error
- Reliability buckets
- Feature drift
- Confidence drift
- Model drift
- Data quality
- Provider health
- Version comparisons
- Champion / Challenger / Shadow diagnostics
- Readiness gates
- Trust components
- Blockers

These diagnostics are read-only and do not mutate models or recommendations.
